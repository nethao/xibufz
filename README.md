# xibufz WordPress Classic Theme

`xibufz` 是从现有静态首页原型重构而来的 WordPress 经典主题，可直接放入 `/wp-content/themes/xibufz/` 启用。

## 安装

1. 将整个 `xibufz` 目录复制到 WordPress 的 `/wp-content/themes/`。
2. 进入后台「外观」->「主题」，启用 `xibufz`。
3. 进入「外观」->「菜单」，创建并分配「主导航菜单」和「页脚菜单」。
4. 进入「外观」->「自定义」->「西部法制网主题设置」，配置首页 Banner、便民服务、友情链接和页脚信息。
5. 发布文章并设置分类、特色图像。置顶文章会优先作为首页主头条。

## 建议创建的分类

- featured
- 公告公示
- 专题专栏
- 综合信息
- 法制热点
- 法律法规
- 法治理论
- 普法宣传

## 首页数据规则

- 头条区优先读取置顶文章 1 篇，次级头条读取最新文章 3 篇。
- Banner 使用自定义器配置；未配置图片时显示主题占位视觉。
- 热门文章优先读取 `featured` 分类最新 5 篇；分类不存在时回退最新文章。
- 公告信息读取「公告公示」最新 4 篇。
- 专题推荐读取「专题专栏」最新 3 篇。
- 独家阅读读取最新文章 3 篇，显示缩略图、标题和摘要。
- 资讯专题模块按指定分类分别显示 1 条带图文章和 4 条列表文章。
- 友情链接、便民服务和页脚信息均可在自定义器中配置。

## 后续增强建议

- 为首页推荐位增加独立排序字段。
- 增加栏目页专用模板和面包屑导航。
- 增加备案链接、社交媒体链接和投稿页面选择器。
- 增加主题翻译文件与 PO/MO 构建流程。
